Introduction:
This source code aims to implement a file downloader that can download a file from a server using TCP. The source code has been implemented in C language. The development environment used for this code was Visual Studio Code. It was compiled using gcc compiler of the Mac OS X.

Steps for compilation:
1. Connect 2 laptops using Wi-Fi ad hoc network.
2. Copy the server code in one system and the client code in another.
3. Open the terminal and access the directory containing the source code.
4. On the server system, compile the given server code using the command
    gcc server.c -o server 
5. Run the server code using the command
    ./server
6. Similarly on the client system, compile and run the client.c file using the command 
    gcc client.c -o client 
    ./client
